# myIot----智慧宿舍

## 系统功能
* 系统能够实时监测当前的温湿度以及光照强度，并且能够通过esp8266将数据上传至云端的服务器，保存到数据库中。
* 用户可通过网页或者微信小程序查看实时的温湿度和光照强度也可查看最近的二十条历史记录。同时，用户也可以向开发板发送控制命令来控制继电器和灯泡。
* 除此之外，当光照强度过高或者过低时，继电器会自动开或者关。继电器可与电机相连，这样就可以通过控制继电器的开关来控制窗帘。
